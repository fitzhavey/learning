# Strings

## What are strings?
- letters, special characters, spaces, and digits
- encapsulated in quotation marks or single quotes
	- `hi = "hello there"`
	- `greetings = 'hello'`
- strings can be **concatenated**
	```
	hi = 'hello'
	name = eric
	greeting = hi + ' ' + name
	# 'hi eric'
	```
